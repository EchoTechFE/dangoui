# `@frontend/du-icon`

## Icon组件

## API 设计

### 组件 Props

| Name | Type | Required | Default | 备注 |
| ---- | ---- | -------- | ------- | ---- |
| name | String | true | | icon的名字 |
| extStyle | String | false | '' | |
| extClass | String, Array, Object | false | '' | |
| size | String | false | | 大小，如 24rpx |
| color | String | false | | 颜色，如 var(--color-main)、red、#666) |

### emit 事件

- click

## Example

```vue
<DuIcon name="c2c" />
```
