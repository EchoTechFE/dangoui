# Swiper 轮播

:stage{type="alpha"}

```ts
import { DuSwiper, DuSwiperItem } from '@frontend/dumpling'
```

## 使用说明

- `DuSwiper` 需要有一个确定的宽高（子组件被限制在这个宽高中），在小程序中，你可以通 `ext-class` 或者 `ext-style` 指定
