# Tabs 标签栏

:stage{type="alpha"}

用于在同一个区域切换内容

```ts
import { DuTabs, DuTab } from '@frontend/dumpling'
```

## 说明

1. 在 `right` 插槽中使用 `DuButton` 和 `DuDivider` 组成工具栏
2. 在小程序中使用 `right` 插槽时，如果你传递给 slot 为一个节点并且为 `view`，请务必给这个 `view` 添加一个 `class`
