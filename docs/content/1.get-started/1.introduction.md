# 简介

draft

一个小程序优先的 Vue 组件库

1. uni-app、web 都可以使用
2. API 设计优先于小程序
3. 业务上久经考验
4. 默认就很好看
5. 易于扩展颜色，满足定制需求

在使用的应用：

潮玩族、奇货、米花星球
