# Dropdown 下拉选择器

:stage{type="alpha"}

下拉打开这个抽屉,喜欢的糖果都在里面🍬

```ts
import { DuDropdown } from 'dangoui'
```

## 使用注意

- `value` 为一个数组，即使只有一列
